# Duh/iKee.B iPhone BotNet
This is a academic restoration of the Duh iPhone BotNet of 2009. 

## Requirements
* Mac OS X
* Jailbroken iPhone (I reccomend a 3G)
* xCode/gcc/g++
* VPS with PHP 5.6
